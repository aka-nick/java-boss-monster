# 구현 기능 문서

## 프로그램의 간략한 흐름1
시작 >> (모두 생존했는가?)[ 플레이어공격 >> 판정 >> 판정에 따른 분기처리 >> 몹공격 >> 판정 >> 판정에 따른 분기처리(반복문의 분기조건을 통한) ] >> 딜교환결과출력 >> 종료

## 프로그램의 간략한 흐름2
시작 >> 보스몬스터정보 입력 >> 플레이어정보 입력 >> 게임진행(게임진행 입력 및 출력. 흐름1 참고) >> 성공/실패출력 >> 종료

## 구현할 기능 목록
- 몬스터 정보 입력
  - HP 입력 기능
- 플레이어 정보 입력
  - 이름
  - HP,MP
- 게임 진행 기능
  - 전원 생존 판정
  - 플레이어 공격
    - 공격방식 택일(물리||마법)
  - 몬스터 공격
    - 데미지 산출(0~20 중 랜덤)
  - 딜교환 결과 알림
- 최종 결과 알림(성공||실패)
